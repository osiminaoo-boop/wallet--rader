HOW TO USE
▶ セットアップ（初回）
1. helius.dev で無料登録してAPIキーを取得
2. 画面上部にAPIキーを貼り付け
3. 監視したい取引所をSCAN TARGETSでONに
4. STARTを押して監視開始
▶ 基本操作
START/STOP — 自動監視の開始・停止
SCAN — 過去10分/30分を手動スキャン
CLEAR — 全データリセット
▶ 見るべきシグナル
🎨 COIN CREATED — 新規ウォレットがPump.funでコイン作成
🎯 TOKEN BUY — 複数の監視ウォレットが同じCAを購入
🏦 OUTFLOW — 取引所から新規ウォレットへ出金を検知
▶ 各パネル
LIVE EVENTS — リアルタイム検知ログ
WATCHED WALLETS — 監視中ウォレット一覧（タップで名前編集）
CREATED COINS — 作成されたコイン一覧
CA RANKING — 複数人が買ったコインのランキング
CA ANALYZER — CAを入力してホルダー数・MC・作成時刻を調査
▶ 注意点
• iPhoneはバックグラウンドで止まることがある → 画面オンのままに
• 無料Heliusは月100万リクエスト上限
• 監視ウォレットが増えすぎるとAPIを多く消費する
