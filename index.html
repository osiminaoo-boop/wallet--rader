HOW TO USE
▶ セットアップ（初回）
1. helius.dev で無料登録してAPIキーを取得
2. 画面上部の「HELIUS API KEY」欄にキーを貼り付け
3. SCAN TARGETSで監視したい取引所を1つ選択
4. STARTを押して監視開始
```
▶ ボタン説明
▶ START — 監視開始。押した瞬間以降に取引所から初めてSOLを受け取った新規ウォレットのみ検知する

■ STOP — 監視停止。再度STARTすると新しいセッションで再開

🔍 SCAN（10分/30分） — 過去10分または30分を遡って手動スキャン。STARTしていなくても使用可能

✕ CLEAR — イベント履歴・統計をリセット。監視ウォレットは保持される

👁 監視リセット — 監視ウォレットリストを全削除
▶ 検知条件
• 選択した取引所からの 0.01 SOL以上 の出金TX
• 出金先のウォレットが 初めてSOLを受け取る（preBalance = 0）
• 出金先がpump.fun / Bags銘柄を 購入 or 作成 したらイベント表示
▶ イベントの種類
🏦 OUTFLOW + NEW WALLET
　取引所から新規ウォレットへの出金を検知。監視リストに自動追加

🎨 COIN CREATED
　監視ウォレットがPump.funまたはBagsでコインを新規作成

🎯 TOKEN BUY
　監視ウォレットがPump.fun / Bags銘柄を購入（CA末尾がpumpまたはBAGSのもの）
▶ 各パネル
LIVE EVENTS — リアルタイム検知ログ
WATCHED WALLETS — 監視中ウォレット一覧。タップで名前編集・手動追加も可
CREATED COINS — 監視ウォレットが作成したコイン一覧
CA RANKING — 複数の監視ウォレットが買ったコインのランキング（20秒更新）
CA ANALYZER — CAを入力してホルダー数・MC・チャートを表示
▶ 注意点
• 取引所は 1つだけ 選択（複数選択不可）
• iPhoneはバックグラウンドで止まる → 画面オンのまま使用
• MEXCはSOL出金が少ない場合あり（USDC出金は非対応）
• 無料Heliusは月100万リクエスト上限。監視ウォレットが多いほど消費が増える
```
